# Bucheen Tasarım P.
Sadece Frontend kısmı

## Kullandıklarım

- *React (Vite la)*
- *Tailwind CSS (v4):*

## Çalıştırma

1. Terminalde dosyaya girip

2. Gerekli paketleri indirmek için bu komut çalışmalı:
   npm install
3. Projeyi ayağa kaldırmak için:
   npm run dev
4. Böylece tarayıcıda `http://localhost:5173/` proje görülebilir durumda.
